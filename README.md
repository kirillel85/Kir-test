Итоговое задание  №2

1. Создайте облачный репозиторий и укажите создание readme.md

2. Склонируйте его себе на компьютер

0 - незачет с комментариями от преподавателя.

1 - зачет, но можно было лучше.

2 - отлично, без замечаний.
